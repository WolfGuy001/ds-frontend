# ds-frontend

Экспериментальный фронтенд для инференса [DiffSinger](https://github.com/openvpi/DiffSinger) — нейросетевого синтеза певческого голоса.

**Текущий статус:** универсальный инференс-пайплайн, работающий с любой DiffSinger-совместимой моделью.

## Что умеет

- Загружать голосовые модели трёх архитектур (flat, mixed, modern modular)
- Прогонять полный 7-шаговый пайплайн: Linguistic → Duration → Pitch → Variance → Acoustic → Vocoder
- Автоматически адаптироваться под конфигурацию модели (флаги, формат фонем, языки, спикеры)
- Сохранять результат в WAV 44100 Hz 16-bit

## Быстрый старт

```bash
npm install
node run-all.mjs
```

## Архитектура

```
src/
├── voicebank.mjs   — загрузчик моделей
├── pipeline.mjs    — инференс-пайплайн
└── utils.mjs       — хелперы тензоров и WAV
```
